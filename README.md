#AirBNB clone 
## Technologies
Project is created with:
* javascript
* EJS
* MongoDB
* Vscode

## Setup
To run this project, install it locally using npm:
```
$ npm init -y
$ npm i express
$ npm i mongosh
$ npm i ejs
$ npm i method-override
$ npm i nodemon
```

#### Run the project
* node index.js or
* nodemon
